## 🏃 How to Run the Programs

You can execute any of these practice programs locally on your computer. Open your terminal and run the following commands:

```bash
# 1. Clone this repository
git clone https://github.com

# 2. Move into the specific practice folder (e.g., day01)
cd day01

# 3. Compile the Java file into computer bytecode
javac Main.java

# 4. Run the program to see the output
java Main
```
