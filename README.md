## 🏃 How to Run the Code

To execute any of the daily programs locally, clone this repository and use the following standard commands in your terminal:

```bash
# Clone the repository
git clone https://github.com

# Navigate to the specific day's folder
cd day01

# Compile the Java file
javac Main.java

# Run the application
java Main
```
